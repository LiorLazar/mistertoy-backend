MisterToy Backend API

Welcome to the MisterToy backend server!

This is a REST API server for the MisterToy application.

Available API Endpoints:

  • GET /api/toy - Get all toys
  • GET /api/toy/:id - Get toy by ID
  • POST /api/toy - Add new toy
  • PUT /api/toy/:id - Update toy
  • DELETE /api/toy/:id - Delete toy
  • POST /api/auth/signup - User signup
  • POST /api/auth/login - User login
  • POST /api/auth/logout - User logout
  • GET /api/user - Get all users
  • GET /api/review - Get all reviews
